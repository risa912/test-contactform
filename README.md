## 環境構築
Dockerのビルド

1. git clone リンク
2. docker-compose up -d
3. docker-compose exec app php artisan migrate
4. docker-compose exec app php artisan db:seed

## 使用技術

- PHP 8.x
- MySQL 8.x
- Docker, Docker Compose
